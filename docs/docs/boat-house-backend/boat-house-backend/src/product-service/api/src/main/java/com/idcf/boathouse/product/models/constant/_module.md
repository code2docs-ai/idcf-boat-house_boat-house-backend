# 说明

OrderConstant类用于定义订单状态的映射关系，并提供一个方法用于获取订单状态的描述。该类的主要功能是将订单状态与其对应的描述信息关联起来，便于在系统中统一管理和使用订单状态信息。通过此类，可以方便地查询和获取订单状态的详细描述，确保订单状态在系统中的一致性和可读性。
