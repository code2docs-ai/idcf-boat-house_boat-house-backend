# 说明

Swagger配置类用于启用Swagger2框架，配置API的基本信息，如标题、描述、版本等，并指定需要扫描的包路径以自动生成API文档。通过该配置，开发者可以方便地管理和展示API接口，提升开发效率和文档的可读性。
